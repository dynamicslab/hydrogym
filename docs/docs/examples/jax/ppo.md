# PPO Training

Pure-JAX PPO training for the Kolmogorov and turbulent channel environments, based on [purejaxrl](https://github.com/luchris429/purejaxrl/) with HydroGym integrations (`VecEnv`, normalization wrappers, etc.).

See the runnable example under `examples/jax/getting_started/3_ppo/` (`run_ppo.py`, `run_ppo_docker.sh`). For a quick command, use the [Getting Started](./getting_started) Docker workflow with `--env kolmogorov` or `--env channel`.
